# Active-Learning
・学習者が受動的に情報を受け取るのではなく、自分自身で問いかけ、課題に取り組み、経験を通じて知識やスキルを獲得することを促進する教育手法のこと<br>
・教師が主に講義を行い、学生が受動的に聴講する形式の授業。教師が教材を用いて知識や理論を解説し、学生はそれをノートに取ったりメモを取ったりしながら聴講する講義型とは異なる教育方法<br>
 <br>
 <br>
### Active learningの定義
・教員による一方向的な講義形式の教育とは異なり、学修者の能動的な学修への参加を取り入れた教授･学習法の総称。学修者が能動的に学修することによって、認知的、倫理的、社会的能力、教養、知識、経験を含めた汎用的能力の育成を図る<br>
・発見学習、問題解決学習、体験学習、調査学習等が含まれるが、教室内でのグループ・ディスカッション、ディベート、グループ・ワーク等も有効なアクティブ・ラーニングの方法<br>
中央教育審議会.2012<br>
・一方向的な知識伝達型講義を聴くという（受動的）学習を乗り越える意味での，あらゆる能 動的な学習のこと」であり，「能動的な学習には，書く・話す・発表するなどの活動への関与と， そこで生じる認知的プロセスの外化を伴う
溝上.2012<br>
 <br>
### Active learningの歴史<br>
・米国の高等教育において1980年代に提起<br>
・ボンウェルとアイソン（Bonwell & Eison, 1991）によって定義され概念化<br>
・日本でアクティブラーニングがはじめて文部科学省の施策用語として登場したのは、いわゆる質的転換答申（中央教育審議会『新たな未来を築くための大学教育の質的転換に向けて（答申）』2012年<br>
・元来、高等教育のために利用されていたアクティブラーニングの初等中等教育学習指導要領の改訂に盛り込まれたのが2014年<br>
・2020年に次期学習指導要領からアクティブラーニングとの用語が消え、「主体的・ 対話的で深い学び」に変更<br>
 <br>

### Active learningの幅と深さ<br>
自由度と創造性、活動の範囲<br>
総合的思考：知識を総合的に利用することで、現実的状況を再現・提示し、学習を進めていく<br>
想像と知識の活用：知識の活用から創造性を構築していく学習であり、個・組織の双方を利用し、現実的状況を再現または現場で行う学習<br>
知識の定着：知識の定着を目的とし、個で学習を進めていく。主に講義形式に近い授業が該当する<br>
意見交換・表現：数名のチームで意見を交換・表現を構築、さらに全体で発表するなど活動範囲を広げた学習<br>

# やるきの科学<br>

## Active-Learningの具現化<br>
Active-Learning　という言葉だけが一人走りしている状態は注意をする必要がある。つまり、教育をActive-Learningに寄せる必要はない。講義形式を利用しつつ、Active-Learningを取り入れるなど、Active-Learningは目的に応じて選択され、あくまでも手段の一つである。必然的に受講する側のActiveさも必要となる。そのためには、環境と個人の２つの観点から受講する側のActiveさを引き出す努力も必要となる<br>

## 個人因子として重要なことは<br>
・教員による一方向的な講義形式の教育とは異なり、学修者の能動的な学修への参加を取り入れた教授･学習法の総称。学修者が能動的に学修することによって、認知的、倫理的、社会的能力、教養、知識、経験を含めた汎用的能力の育成を図る<br>
**少なからず個人のやる気に依存するものもある**<br>

## ROEE Results Only Education Environment
・「結果のみが求められる職場環境(Work)や教育環境(Education)」のこと<br>
・すなわち、「ROEE」は学生にはやらなければならない課題を提示し、学生自身に課題達成の過程に責任を与えることで、結果を出すための自主性を持たせることを意識させる<br>
・スケールは、研究室やゼミなど小グループや個人を対象とした小さな単位の課題もあれば、集団に対してもクリアする課題を明確化し、学生自身が考え、主体的に動くことができる環境を提供する<br>
・最終的な責任の所在は明確にし、教育であれば教員が与える課題の難易度を学生個人に合わせて考慮することで学生自身が持つべき責任の大きさを明確すること<br>

**つまり、教育には責任が伴うことを理解すること**<br>
**そして、如何に自主性を持たせることでやる気にさせること**<br>

## どのようにやる気にさせるのか？ Candle problem
> Karl Duncker  1945<br>
 <br>
問い：テーブルに蝋が垂れないように壁に蝋燭を取り付けてください<br>


機能性固着:ものの機能（用途）に固着するあまり、そのほかの機能に気づかなくなることで課題の難易度が高まる<br>
 <br>
 <br>
 
> Sam Glucksberg 1965<br>
 <br>
問い：テーブルに蝋が垂れないように壁に蝋燭を取り付けてください。この問題をどれくらい早く解けるか時計で計ります<br>
1)平均時間を知りたい<br>
2)報酬を提示<br>

**報酬を提示のグループが3分半余計にかかった**<br>
**つまり、インセンティブは効果的ではない**<br>

→後者が3分半余計にかかった：インセンティブは効果的ではない（創造性においては）<br>
If Then式に 「これをしたら これが貰える」というやり方は状況によっては機能しないことを証明している<br>

 
## どのようなときにやる気にさせるのか？ 
それではどのようなケースでIf Then式、つまり報酬が効果的になるのか<br>
同様の実験を少しアレンジしてやってみた→機能性固着の撤廃を行うことで、問題を単純化した<br>
 
この結果、報酬を提示のグループが早い結果となった<br>
単純な問題ほど、インセンティブが効果を発揮し、創造性の課題にはインセンティブは効果を発揮しない<br>
課題難易度よって動機付けを変化させる必要がある<br>
<br>
<br>

> Edward L. Deci 1965<br>
　<br>

  反転授業
  従来の授業形式とは異なり、教師が授業で講義をする前に、生徒が自宅や学習センターなどで教材や動画を事前に学習し、授業時間にはその理解度を深めたり、実践的な活動を行う形式の授業
  具体的には、教師が授業で取り上げるテーマや内容に関する動画や教材を事前に生徒に提供し、自主的に学習してもらいます。
  そして、授業時間には教師が生徒たちに指導したり、グループワークやディスカッション、実験などを行い、より実践的な学びを促します。
  
  反転授業のメリットとしては、生徒が授業に参加する前に学習することで、授業時間をより有意義に使えることや、自己学習能力が向上することが挙げられます。
  また、自分のペースで学ぶことができるため、理解度が深まることも期待できます。

一方で、反転授業では、生徒たちが自主的に学習するために、指導力のある教師や十分な学習環境が必要であるという課題もあります。
また、生徒たちが自分のペースで学ぶことができるため、授業の進行が遅くなることもあるため、十分な計画性が必要となります。
